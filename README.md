# 🎓 Jom Pandai Academy: Misi Wira Ilmu

Permainan web interaktif (prototaip) yang direka khas untuk pelajar sekolah rendah (Darjah 1-6). Berkonsepkan pengembaraan (adventure RPG), permainan ini menukar latihan menjawab soalan matematik menjadi satu misi menyelamatkan dunia!

## 📖 Jalan Cerita
Seekor naga gelap telah mencuri 6 Kristal Ilmu. Pemain perlu menjelajah dunia, menjawab soalan, mengalahkan musuh, dan menyelamatkan setiap dunia.

## 🎮 Cara Bermain (Prototaip Semasa)
- **Serangan:** Jawab soalan matematik dengan betul untuk membolehkan Hero menyerang musuh.
- **Bertahan:** Jika jawapan salah, musuh akan menyerang dan Hero akan kehilangan 1 Nyawa (❤️).
- **Ganjaran:** Dapatkan 10 Coin (🪙) untuk setiap jawapan betul. Bonus 50 Coin jika berjaya jawab 5 berturut-turut!
- **Level Up:** Kumpul coin untuk naik level (soalan akan menjadi lebih mencabar).
- **Game Over:** Jika 5 Nyawa habis, permainan tamat.

## 🚀 Perancangan Masa Depan (Berdasarkan Game Design Document)
Versi penuh akan merangkumi:
- 🌍 **6 Dunia Misi:** Hutan Nombor, Padang Pasir Matematik, Gunung Sifir, dll.
- 🐉 **Boss Battles:** Lawan Boss besar setiap 20 soalan.
- 🦄 **Sistem Pet & Skin:** Beli skin hero dan pet menggunakan Coin.
- 🏆 **Pencapaian & Misi Harian:** Lucky Spin dan cabaran harian.
- 👥 **Multiplayer:** Mod Duel (1v1 skrin sama) dan Squad.

## 🛠️ Teknologi Digunakan (Prototaip)
- HTML5
- CSS3 (Animasi Flexbox & Keyframe)
- Vanilla JavaScript (Logik Game & Penjanaan Soalan Math)
